Detailed Description

Vibrate_manager is a singleton that manages the logic for vibrating and prevents the app from vibrating more than once per second.

Function Documentation

void vibrate_manager_init ( )

Initialize the Vibrate_manager.

Should be called once when the app is started and before any other Vibrate_manager methods are called.

void vibrate_manager_uninit ( )

Tear down the Vibrate_manager.

Should be called once when the app is closing. Vibrate_manager methods should not be used after this is called.

Use the Group's Settings and Timer to determine if the app should vibrate.
