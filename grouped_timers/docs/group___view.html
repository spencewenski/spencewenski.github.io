Detailed Description

An abstract view type that provides a fat interface for all the different views.

This allows the Model to interact with all the views without needing to know the exact implementation of any specific view. This makes it very easy to add new types of views without modifying the Model. Views can set the fields of View_interface to point to the implementations of the callbacks.
