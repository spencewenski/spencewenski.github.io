Detailed Description

Methods that can be implemented to extend the abstract View.

If a method returns int, the return value indicates whether the View should be refreshed at the next refresh. A positive return value indicates that the method made the View dirty and that it should be refreshed, while a return value of zero indicates that the method didn't modify the View and/or the View doesn't need to be refreshed.
