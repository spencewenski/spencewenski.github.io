void timer_increment_field ( struct Timer *  timer,
enum Timer_field  field,
int  amount 
)

Increment the Timer_field by the given amount.

The provided amount is in the same units as the Timer_field. This method bounds the value to a [min, max) for each field. If the field is at a bound and an increment would take the field out of [min, max), the field is 'looped' to the other bound. If the field is not at a bound and an increment would take the field out of [min, max), the field is set to the respective bound.

Parameters
timerTimer to edit.
fieldTimer_field to edit.
amountAmount to edit the Timer_field by. Should be in the same units as the Timer_field, e.g. 1 to increment the TIMER_FIELD_HOURS by one hour.
