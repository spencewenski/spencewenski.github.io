Detailed Description

Utilities to help with drawing on the screen.

These are some utilities and commonly used functions used for drawing graphics on the screen.

enum Preferred_font_size

Abstract font sizes to use in the app.

Used to define the maximum preferred font size for dynamically setting the font size of a string.

Enumerator
